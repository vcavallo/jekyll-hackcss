Jekyll gem-theme based on hack.css

This theme is more-or-less wemake-services/jekyll-theme-hackcss but gemified so people can install it on their existing jekyll sites.

What is hack.css?

Head over to the hackcss site to get the official word.
Briefly, it's a simple css framework for markdownophiles.
